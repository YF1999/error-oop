# CHANGELOG
## [v0.2.0] node-errors-pro (2020-01-10)
### Features

- Support more usefull errors:
    - IndexOutOfRangeError
    - OperationCanceledError
    - UnauthorizedAccessError
    - UnauthorizedError

## [v0.1.0] node-errors-pro (2021-01-08)
### Features

- Support native errors pro:
    - ErrorPro
    - RangeErrorPro
    - ReferenceErrorPro
    - SyntaxErrorPro
    - TypeErrorPro
    - URIErrorPro

- Support some more usefull errors:
    - AlreadyInUseError
    - ArgumentError
    - ArgumentNullError
    - ArgumentOutOfRangeError
    - AuthenticationError
    - AuthenticationRequiredError
    - ConnectionError
    - InvalidOperationError
    - NotFoundError
    - NotImplementedError
    - NotPermittedError
    - NotSupportedError
    - OutOfMemoryError
    - StackOverflowError
    - TimeoutError
    - Data
        - DataError
        - TransactionError
    - IO
        - IOError
        - DirectoryNotFoundError
        - DriveNotFoundError
        - EndOfStreamError
        - FileLoadError
        - FileNotFoundError
        - PathTooLongError
        - SocketError

- Support create custom errors by inheriting the existing errors
